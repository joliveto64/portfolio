A for-fun project built using the P5 library, a JavaScript library with a focus on making coding accessible and inclusive for artists and designers. As the name states, this project randomly generates (moving) clouds!

Theme Generator

Using The Color API, this project generates a color palette based on a selected color and scheme (such as monochrome, complement and analogic among others). Both RGB and HEX values can be copied on click! This app is responsive.

Technologies: HTML, CSS, JavaScript, The Color API

Interactive Fireworks

Another for-fun project built with P5! This one is a bit more complex than the cloud generator, featuring fireworks of random size and color that launch along whichever vertical axis you touch (or click). The cityscape is randomly generated with every refresh and the fireworks leave smoke behind!

Technologies: JavaScript, P5.js library

Hunger Scale

Using data from the Calorie Ninjas API, this project generates values to represent satiety (level of satisfaction) for any food available in the database. By comparing different foods, you can identify which ones are more likely to keep you fuller, longer! This app is responsive.

Technologies: HTML, CSS, JavaScript, React library, Calorie Ninjas API
